What is CodeLingo? 
-
CodeLingo is a Duolingo-style syntax trainer that was made to help students strengthen their programming foundation through fast-paced, gameified repition.
Instead of passively reading syntax, players can actively test their knowledge accross multiple programming languages while tracking score, XP, and streaks, and speed.

Purpose
-
Learning syntax could be boring and repetative. CodeLingo turns syntax practice into an interactive quiz experience that rewards speed, consistency, accuracy, and high scores
This project was built as a front end web app to reinforce core programming concepts in afun and competative format. 

Supported Languages 
-
C++, Python, Java, Javascript
Each language contains 10 randomized multiple choice questions per session

Features 
-
Game like quiz, randomized question and answer, XP system with speed bonus, Streak tracking, Confetti animation for perfect scores, keyboard shortcut 1-4 to answer + enter for next, and progress bar

Tech Stack 
-
HTML5, CSS, Javascript

How to run 
-
Clone repo onto your Computer. Open index.html in your browser with visual studio's live server extension 

Potential Future improvements 
-
Difficulty levels, Timed mode, More languages, Mobile, and sound effect + animations.
